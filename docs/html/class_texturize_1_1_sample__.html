template<size_t cn>
Texturize::Sample_ class

A class that provides unified access to image samples.

Template parameters
cn The number of channels of the sample.

Contents

  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

This class directly inherits from Sample and provides the same semantics. It is designed for cases, where the number of channels of a sample is known at compile time. For example, UV maps allways require a set of 2 channels. In such cases, the template can be used to efficiently pre-allocate memory.

template<size_t cn> template<size_t _sk>
void Texturize::Sample_<cn>::getNeighborhood(const int x, const int y, cv::Vec<float, _sk*cn>& v, const bool weight = false) const

Returns the neighborhood of a texel.

Template parameters
_sk The size of the neighborhood window. Must be an odd value.
Parameters
x The x coordinate of the texel.
y the y coordinate of the texel.
v The buffer to copy the neighborhood to.
weight A boolean value that toggles gaussian weighting of neighborhood values, depending on the distance to the texel identified by the provided coordinates.

If the pixel coordinates address a texel at the border of the sample, the method will automatically wrap the coordinates. Other border modes are currently unsupported. The result texel v is a high-dimensional vector, that contains all neighborhood values in row-major order. If the sample has multiple channels, all channels are copied, before moving to the next pixel.
