Texturize::ISearchSpace class

An interface that defines methods to create, transform and work with search spaces.

Contents

  • Reference
    • Derived classes
    • Public functions

Basically, a search space is the result of image analysis. It is used later during synthesis in order to match pixel or patch neighborhoods. This interface describes the base for different possible search space implementations. Naive sampling approaches match pixel neighborhoods based on their color values, so there is not much analysis required. More advanced approaches utilize complex search space implementations.

The search space itself describes pixels and pixel neighborhoods as a Sample. Synthesis algorithms do not directly access the search space, but instead have to create an index over the search space. The details of this process is described under Texturize::ISearchIndex.
