Texturize::CoordinateHash class

Generates a permutation vector from a set of coordinates.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Different to random noise functions, like gaussian or perlin noise, the CoordinateHash returns the same offset for a indentical set of input coordinates, hence the name hash. It can be used to create reproduceable noise in parallel algorithms.
